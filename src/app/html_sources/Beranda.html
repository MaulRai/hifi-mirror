Akademik

Pendidikan Sarjana

Fasilkom UI menyelenggarakan program studi sarjana ilmu komputer dan sistem informasi. Khusus program studi ilmu komputer, tersedia kelas internasional bergelar ganda.

Pendidikan Pascasarjana

Untuk jenjang pascasarjana, Fasilkom UI menawarkan tiga program studi, yaitu:  Magister Teknologi Informasi, Magister Ilmu Komputer, dan Doktor Ilmu Komputer.

Penerimaan Mahasiswa Baru

Fakultas Ilmu Komputer Universitas Indonesia menerima mahasiswa baru untuk semua program studi pada jenjang sarjana, magister, dan doktoral.

Laboratorium Riset

Kegiatan riset Fasilkom UI didukung oleh tujuh laboratorium, yaitu Computer Systems; Information Retrieval & Natural Language Processing; Reliable Software Engineering; Digital Library & Digital Learning; E-government & E-business; Machine Learning & Computer Vision; dan Information Management.

Pusat Penelitian

Khusus bagi mahasiswa dan peneliti yang ingin mengembangan inovasi di bidang kecerdasan buatan dan keamanan siber, Fasilkom UI difasilitasi oleh tiga pusat penelitian berbasis industri yaitu Tokopedia-UI Artificial Intelligence Center of Excellence, Cyber Security & Cryptography Center, dan Lexin – UI Center for Legal Informatics.

Jurnal Ilmiah

Hasil penelitian dan inovasi Fasilkom UI tidak hanya diterbitkan di tempat publikasi bertaraf internasional, tetapi juga jurnal-jurnal fakultas sendiri. Dua jurnal Fasilkom UI telah dikenal menjadi barometer kualitas publikasi penelitian nasional di bidang ilmu komputer, teknologi informasi, dan sistem informasi.
